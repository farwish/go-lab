## 模块化
`export GO111MODULE=on` 开启 golang 模块化包管理，将不再使用 GOPATH 管理依赖.
`go mod init /home/wc/go-lab/go-syntax` 用来初始化模块目录.
go.mod 不能在 GOPATH 目录下，go.sum 是当前模块依赖项目的版本锁定.
